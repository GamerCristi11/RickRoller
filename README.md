## Installation
1. Download the `.crx` file from this repo
2. In the **Extensions Manager** (`chrome://extensions`)
3. Enable **Developer Mode**
4. Drag and drop the downloaded `.crx` file into the **Extensions Manager**
5. Confirm any prompts to install the extension.
6. **Have fun! :)**
