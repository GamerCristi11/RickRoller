# RickRoll Redirector
Prank your friends with this simple extension that redirects any YouTube videos to a RickRoll video

## Installation
1. Download this repo
2. In the extensions manager, enable **Developer mode**
3. Click **Load Unpacked** and select the downloaded folder
4. Have fun :)
